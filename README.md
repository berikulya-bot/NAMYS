# NAMYS# EcologicAI | Sentinel Dashboard

## Проект для SynergyX Hackathon
Умная система прогнозирования экологических рисков и мониторинга микропластика.

### О проекте
EcologicAI — это высокотехнологичная платформа для предсказательного мониторинга качества воды. 
Мы используем данные геолокации и спутниковые слои, чтобы выявлять зоны высокого риска загрязнения микропластиком.

### Основные функции
- 🛰 **Satellite Integration:** Интеграция спутниковых слоев Google Maps.
- 📍 **Real-time GPS Tracking:** Отслеживание зоны мониторинга в реальном времени.
- 🧠 **Predictive AI:** Имитация глубокого анализа данных (Soil Permeability + Hydrology).
- 📱 **Enterprise UI:** Профессиональный интерфейс в стиле Glassmorphism.

### Стек технологий
- Frontend: HTML5, Tailwind CSS
- Maps: Leaflet.js
- Data: OpenStreetMap, Google Satellite Tiles
